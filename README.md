# 🧪 API Testing Project - Restful Booker

## 📌 Overview
This project demonstrates automated API testing using **Postman** and **Newman** on the Restful Booker API.  
It validates multiple endpoints, checks response correctness, and measures performance.

---

## 🚀 Tools & Technologies
- Postman (API testing & collection creation)
- Newman (CLI runner for Postman collections)
- Node.js
- HTML Reporter (Newman)
